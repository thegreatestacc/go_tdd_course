## Оценка качества

Оценка отражает текущее состояние решений, а не только факт прохождения
существующих тестов.

| Критерий | Балл | Комментарий |
|---|---:|---|
| Корректность | 31 / 40 | Основные сценарии работают; `Repeat` игнорирует аргумент `count`, а неизвестный язык в `Hello` не использует английское приветствие |
| Тесты | 24 / 30 | Есть subtests, test helper и проверка пустого tail; не хватает тестов, обнаруживающих указанные дефекты |
| Go-стиль | 17 / 20 | Код проходит `gofmt`, используются `strings.Builder`, `slices.Equal` и idiomatic range |
| Структура проекта | 6 / 10 | В `go.mod` указана минимальная версия Go 1.24, необходимая для `b.Loop()`; большинство реализаций пока находятся в файлах `_test.go` |
| **Итого** | **78 / 100** | Все существующие тесты проходят; перед следующими главами стоит исправить найденные дефекты и структуру файлов |

## Проверка

Команда для запуска всех решений:

```powershell
go test ./solutions/...
```

Все пять пакетов проходят тесты на Go 1.26.4. Минимальная версия проекта —
Go 1.24, поскольку benchmark использует добавленный в этой версии `b.Loop()`.

## Как считается прогресс

В основную программу включены практические главы из разделов
**Go fundamentals**, **Testing fundamentals** и **Build an application**:
всего 31 глава. Установочные, справочные и meta-главы в процент прогресса
не входят.
